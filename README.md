This is an App for finding lost lego sets that I'm developing with the help of AI. 
It lets you type in the set number of a lego set and then gives you a parts list you can work through to find all the pieces again.
It uses my rebrickable API key so maybe don't overdo it. 
Thsi app is intended for mobile, but for it to work properly you have to add it to your homescreen.
